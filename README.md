#  LotHub - O Hub Sobre a Névoa

> "O lugar que não pertence a esta era. O misterioso governante acima da névoa cinza."

Este é o repositório oficial do front-end da **LotmHustbi**, servindo como portal central (Launcher Web) para os servidores de jogos da empresa.

## Sobre o Projeto

O **LotHub** é uma Landing Page imersiva projetada para unificar o acesso aos servidores de MMORPG clássicos mantidos pela organização. A estética visual é fortemente inspirada na obra _Lord of the Mysteries_ (Steampunk, Vitoriano, Lovecraftiano, Névoa).

### Caminhos Disponíveis (Servidores)

- **Tibia (The Tyrant):** Servidor Old School 7.4 com mecânicas modernas.
- **PokéWorld (The Visionary):** Servidor de captura e batalha de monstros.
- **Mu Online (The Sun):** _Em desenvolvimento (Selado)._
- **Naruto (The Hunter):** _Em votação._

## Tecnologias (A Fundação)

O projeto foi construído focando em performance e estética pura, sem dependência de frameworks pesados nesta etapa inicial.

- **HTML5 Semântico:** Estrutura robusta e acessível.
- **CSS3 Moderno:** Uso de CSS Variables (`var(--gold)`), Flexbox e Grid Layout.
- **Vanilla JavaScript:** Para interações leves (Menu Mobile).
- **FontAwesome:** Ícones vetoriais.

## Como Executar (Invocação)

Como este é um projeto estático por enquanto:

1. Clone o repositório:
   ```bash
   git clone https://github.com/SeuUsuario/LotHub.git
   ```
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. (Opcional) Use a extensão "Live Server" no VS Code para desenvolvimento em tempo real.
